Linux常用命令及使用方法

2018/11/25 posted in  学习

Linux是一款非常不错的操作系统，适合应用在服务器上，谁用谁知道。以下是一些常用的基础命令及用法：

文件和目录常命令

1 查看目录内容

1.1 查看目录ls

参数 含义
-a 显示指定目录下所有的目录及文件，包括隐藏文件
-l 以列表方式显示文件的详细信息
-h 配合-l以人性化的方式显示

1.2 ls通配符的使用

通配符 含义
* 代表任意个数个字符
？ 代表任意一个字符，至少1个
[] 表示可以匹配字符组中的任一以
[abc] 匹配a、b、c中的任意一个

2 切换目录

2.1 cd

命令 含义
cd 切换到当前用户的主目录
cd ~ 切换到当前用户的主目录
cd . 保持在当前目录不变
cd .. 切换到上级目录
cd - 可以在最近两次工作目录之间来回切换

3 创建和删除操作

3.1 创建文件touch

  • 如果文件不存在，生成一个空白文件
  • 如果文件存在，修改文件的最后一次修改时间

3.2 创建目录mkdir

选项 含义
-p 可以递归创建目录

3.3 删除rm

选项 含义
-f 强制删除，忽略不存在的文件，无需提示
-r 递归地删除目录下的内容，删除文件夹时必须使用此参数

4 拷贝和移动文件

4.1 tree

选项 含义
-d 只显示目录

4.2 拷贝文件cp

选项 含义
-i 覆盖文件前提示
-r 若给出的源文件是目录，则cp将递归复制该目录下的所有目录和文件，目标文件必须为一个目录名

4.3 移动文件mv

  • mv命令可以用来移动文件或目录，也可以给文件和目录重命名
选项 含义
-i 覆盖文件前提示

5 查看文件内容

5.1 查看文件cat

选项 含义
-b 对非空输出行编号
-n 对输出的所有行编号

5.2分屏显示 more

  • more可以用于分屏显示文件内容，每次只显示一页内容
操作键 功能
空格键 显示手册页的下一屏
enter键 一次滚动一页
b 回滚一屏
f 前滚一屏
q 退出
/word 搜索word字符

5.3 查询grep

选项 含义
-n 显示匹配行及行号
-v 显示不包含匹配文本的所有行
-i 忽略大小写
参数 含义
^a 行首，搜索以a开头的行
a$ 行尾，搜索以ke结束的行

6 其他

6.1 显示参数echo

  • 显示指定的文字，通常和重定向联合使用

6.2 重定向 >和>>

  • 将命令执行结果重定向到一个文件
  • 将本应显示在终端上的内容输出/追加到指定文件中

6.3管道 |

  • 将一个人命令的输出可以通过管道作为另一个命令的输入
    常用的管道命令有：
  • more:分屏显示内容
  • grep：在执行结果的基础上查询指定的文本

远程管理常用命令

1 关机重启

1.1 shutdown

 选项 含义
不指定 表示1分钟之后关闭电脑
-r 重启
-r now 现在重启
shutdown 19:00 19:00关机
shutdown +10 再过十分钟后关机
shutdown -c 取消关机

2 查看或配置网卡信息

2.1 查看网卡信息 ifconfig

  • ifconfig | grep inet 查看网卡对应的ip地址

2.2 ping

  • ping ip 和Windows的cmd用法一样

3 远程登录和复制文件

3.1 远程登录ssh

ssh [-p port] user@remote
  • user 远程机器上的用户名
  • remote远程机器上的地址，域名或ip或别名
  • port 端口号默认22

3.2 远程复制scp

scp -P port [源文件] user@remote：绝对路径 [目标文件]

用户权限相关命令

1 用户管理

1.1 创建用户密码和删除

命令 含义 说明
useradd -m -g 创建用户 -m自动建立用户 -g指定用户所属组
password 用户名 设置用户密码 普通用户可以修改账户密码
userdel -r 删除用户 -r选项会自动删除用户目录
cat /etc/password | grep 用户名 确认用户信息 新建用户后，信息会保存在/etc/password文件中

1.2 修改拥有者

chown 用户名 文件名|目录名

1.3 创建组/删除组

  • 需要通过增加权限sudo来执行
命令 含义
groupadd 组名 添加组
groupdel 组名 删除组
cat /etc/group 确认组信息
chgrp -R 组名 目录名/文件名 递归修改所属组

1.4 修改权限chmod

chmod +/-rwx 文件名|目录名

chmod -R 755 文件名|目录名

1.5 查看命令所在位置whitch

which ls 查看ls命令所在位置

系统信息相关命令

1 时间和日期

命令 含义
date 查看系统时间
cal calendar 查看日历 -y 查看一年的

2 磁盘信息

命令 含义
df -h disk free显示磁盘剩余空间
du -h disk usage显示目录下的文件大小

3 进程信息

命令 含义
ps aux a显示所有用户进程，u进程详细状态，x没有控制终端的进程
top 动态显示运行中的进程并排序
kill [-9] 进程号 终止指定代号，-9表示强制终止

其他命令

1 查找文件find

find [路径] -name “关键字”

2 软链接和硬链接ln

ln [-s] 源文件 链接文件

没有-s相当于windows的快捷方式
有-s是一个硬链接，两个文件占用相同的空间

3 打包/压缩tar

3.1 打包

tar -cvf 打包文件名.tar 被打包的文件 /路径

打包+压缩

tar -zcvf 打包文件名.tar 被打包并压缩的文件 /路径

3.2 解包

tar -xvf 打包文件名.tar

解包+压缩

tar -zxvf 打包文件名.tar.gz -C 路径
选项 含义
c 生成档案文件，创建打包文件
x 解开档案文件
v 列出归档解档的详细过程，显示进度
f 指定档案文件名称，必须放最后

4 软件安装更新及卸载apt

安装

sudo apt install 软件包

卸载

sudo apt remove 软件名

更新

sudo apt upgrade
